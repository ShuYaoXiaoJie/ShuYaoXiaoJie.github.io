命令行建立数据库表：

1、连接数据库：

mysql -uroot -proot

2、检测：

set names gbk;

3、复制数据库建立的代码：

DROP DATABASE shopMFY;
CREATE DATABASE IF NOT EXISTS shopMFY;
USE shopMFY;
– 管理员表 –
DROP TABLE IF EXISTS MFY_admin;
CREATE TABLE MFY_admin(
id tinyint unsigned auto_increment key,
userName varchar(20) not null unique,
passWord char(32) not null,
email varchar(50) not null
)ENGINE=InnoDB AUTO_INCREMENT=7 DEFAULT CHARSET=utf8;

– 分类表 –
DROP TABLE IF EXISTS MFY_cate;
CREATE TABLE MFY_cate(
id smallint(5) unsigned auto_increment key,
cName varchar(50) unique
)ENGINE=InnoDB AUTO_INCREMENT=7 DEFAULT CHARSET=utf8;

– 商品表 –
DROP TABLE IF EXISTS MFY_pro;
CREATE TABLE MFY_pro(
id int unsigned auto_increment key,
pName varchar(50) not null unique,
pSn varchar(50) not null,
pNum int unsigned default 1,
mPrice decimal(10,2) not null,
iPrice decimal(10,2) not null,
pDesc text,
pImg varchar(50) not null,
pubTime int unsigned not null,
isShow tinyint(1) default 1,
isHottinyint(1) default 0,
cId smallint unsigned not null
)ENGINE=InnoDB AUTO_INCREMENT=7 DEFAULT CHARSET=utf8;

– 用户表 –
DROP TABLE IF EXISTS MFY_user;
CREATE TABLE MFY_user(
id int unsigned auto_increment key,
usename varchar(20) not null unique,
password char(32) not null,
sex enum(“保密”,”男”,”女”) not null ,
face varchar(50) not null ,
regTime int unsigned not null
)ENGINE=InnoDB AUTO_INCREMENT=7 DEFAULT CHARSET=utf8;

– 相册表 –
DROP TABLE IF EXISTS MFY_album;
CREATE TABLEMFY_album(
id int unsigned auto_increment key,
pid int unsigned not null,
albumPath varchar(50) not null
)ENGINE=InnoDB AUTO_INCREMENT=7 DEFAULT CHARSET=utf8;

每一行命令均出现“Query OK, 1 row affected (0.00 sec)”即算成功。

4、查看表：

show tables;

出现：

mysql> show tables;
+——————-+
| Tables_in_shopmfy |
+——————-+
| mfy_admin |
| mfy_album |
| mfy_cate |
| mfy_pro |
| mfy_user |
+——————-+
5 rows in set (0.00 sec)

5、注意：

第一：表名和字段名用反引号，而不是单引号

第二：注释表的时候–和文字之间至少空出一个空格

6、在shopMFY的文件夹下建立测试用户test.php，在本地localhost进行查看（可使用WampServer）,

注意，要删除本地文件夹内的index.php文件，不然会显示空白。
test.php的内容：
<?php
echo md5("Queen");

$sql = “insert MFY_admin(username,password,email)values(‘Queen’,’b2086154f101464aab3328ba7e060deb’,’419626329@qq.com’)”;
echo $sql;
